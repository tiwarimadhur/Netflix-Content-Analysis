# Netflix Content Analysis (Python & Pandas)

This project analyzes the Netflix Movies and TV Shows dataset using Python, Pandas, and NumPy.  
It includes data cleaning, exploration, and extraction of meaningful insights to understand content trends on Netflix.

---

## Dataset
The dataset contains information about:
- Title  
- Type (Movie / TV Show)  
- Director  
- Cast  
- Country  
- Release year  
- Duration  
- Genre (listed_in)  
- Date added  
- Description
